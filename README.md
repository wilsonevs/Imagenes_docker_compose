# Imagenes_docker_compose

¿Qué es?

Docker Compose es una herramienta para definir y ejecutar aplicaciones de Docker de varios contenedores. 
En Compose, se usa un archivo YAML para configurar los servicios de la aplicación. Después, con un solo comando, 
se crean y se inician todos los servicios de la configuración..
